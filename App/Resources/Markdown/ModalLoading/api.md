#### ATModalToast


| 属性  | 说明         | 类型           | 默认值  |
| ----- | ------------ | -------------- | ------- |
| content | (必填) 提示内容      |  string         | `''`  |
| duration | 持续显示的时间      |  number         | `Theme.modal_toast_duration`  |
| position | 显示位置（默认下方）     |  string         | `bottom`  |
