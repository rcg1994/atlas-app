#### ATLoading


| 属性  | 说明         | 类型           | 默认值  |
| ----- | ------------ | -------------- | ------- |
| size  | 设置大小，`large`、`small` 或者数字(仅安卓) | string、number(android only) | `large` |
| color | 设置颜色      | string         | `null`  |
| title | 设置加载文案   | string         | `null`  |
| indicator | 自定义的图案，暂不支持自动旋转 | any | null |
