#### ATText

| 属性     | 说明                               | 类型 | 默认值 |
| -------- | ---------------------------------- | ---- | ------ |
| size | 字体大小 | number  | `Theme.font_content` |
| color | 字体颜色 | string  | `Theme.font_content_color` |
| lineHeight | 字体颜色 | number  | `Theme.font_line_height` |
| weight | 字重 | string  | `null` |
| style | 字体样式 | any  | `null` |
