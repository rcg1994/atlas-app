#### ATTag

| 属性     | 说明                               | 类型 | 默认值 |
| -------- | ---------------------------------- | ---- | ------ |
| type | 标签类型 | string  | `primary` |
| content | （必填）标签内容 | string  | `null` |
| color | 标签颜色（只支持16进制，背景色计算而得） | string  | `null` |
| icon | 图标 | any  | `null` |
| left | 左边元素 | any  | `null` |
| style | 标签样式 | any  | `null` |
| textStyle | 标签字体样式 | any  | `null` |
