#### ATModalSelect (ATModal.select)


| 属性  | 说明         | 类型           | 默认值  |
| ----- | ------------ | -------------- | ------- |
| buttons | (必填) 按钮选项      |  array         | `''`  |
| bottom | 是否在下方      |  boolean         | `false`  |
| float | 是否悬浮      |  boolean         | `false`  |
| width | 宽度（悬浮时设置）     |  number         | `''`  |
